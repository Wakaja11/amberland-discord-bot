# Discord Bot — AmberLand

Бот для Discord-сервера Minecraft сервера AmberLand.

## Функции
- Подача заявок
- Тикеты поддержки + ИИ-помощник
- Временные голосовые каналы
- Авто-роли
- Статистика
- Анти-спам

## Локальный запуск

```bash
pip install -r requirements.txt
cp .env.example .env
python bot.py
